Route42 Community Edition v0.2.0.

**Local-first LLM router: the right model for every prompt — your GPU first, cloud only when it's worth it.**

## What's new

### Built-in web console
`route42 serve` now hosts a small management console at `http://localhost:4242`, embedded directly in the binary — plain HTML/CSS/JS, no build step, no external assets, no extra download. It mirrors the Route42 Pro look and feel:

- **Dashboard** — gateway health, discovered local models, usage/spend/tokens over time, and copy-paste client snippets.
- **Models** — the full routable catalog with quality/price metrics and availability, plus provider key management.
- **Preferences** — routing priority, model filters, cost/latency limits, and fallback, with a live profile summary.
- **Playground** — type a prompt, watch the routing decision (complexity, category, ranked candidates), and run it with streaming output.
- **Interaction History** — the local request log: model, category, tokens, cost, latency.

The console is **optional**. Community Edition remains fully usable headless via the CLI and HTTP API — set `server.ui: false` (or `ROUTE42_UI=false`) to turn it off. When `server.api_token` is set, the console prompts for the token and sends it on its API calls; `/health` and the static console assets stay public, only `/api/*` and `/v1/*` require the token.

### Other changes
- New `GET /api/interactions?limit=N` endpoint exposes the local interaction log (default 50, max 500) — this powers the History page and is scriptable on its own.
- `/api/prefs` and `prefs set --json` now use snake_case field names (`fallback_depth`, `max_cost_cents`, …), matching the documented wire format. Existing databases are unaffected.

## Install
Download the archive for your platform below, unpack, and run:

    ./route42 serve

Then open `http://localhost:4242` in a browser, or point any OpenAI client at the same address. Zero-config first run works Ollama-only with no API keys.

Prefer no browser UI? `./route42 serve` with `ROUTE42_UI=false` runs headless; everything is available through the CLI (`route42 --help`) and the HTTP API. See the [README](https://github.com/krugis/route42app#readme) for both paths.

Verify downloads against `SHA256SUMS`.
